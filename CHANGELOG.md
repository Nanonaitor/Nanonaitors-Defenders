# Changelog

## 1.1.2 - Guard reliability and Netherite progression

- Made Defender guard immune to axe-style shield disabling.
- Ensured an off-hand Defender can begin guarding while an entity is targeted
  within interaction range.
- Removed the erroneous main-hand swing sound when guarding with a Defender in
  each hand.
- Restored vanilla Netherite progression: combine a Netherite Upgrade Smithing
  Template, Diamond Defender, and Netherite Ingot in a smithing table.
- Preserved attacking, continuous mining, and Forgotten Nunchakus compatibility
  while guarding.

## 1.1.1 - Initial Forge 1.20.1 release

- Ported the complete Defender combat system to Minecraft 1.20.1 and Forge 47.4.18.
- Added Wood, Stone, Copper, Gold, Iron, Diamond, and Netherite Defenders.
- Added conditional Bronze, Steel, Silver, Umbrium, Dragonbone, dragon-blooded,
  Myrmex, Myrmex Stinger, Living, and Sentient variants.
- Added perfect parries, sustained melee guarding, Vulnerable, durability costs,
  configurable movement penalties, and configurable combat values.
- Added Footwork, Reprisal, Finesse, Reflexes, Deflection, and 6th Sense.
- Added configurable enchantment switches and an optional compatible-enchantment
  registry-ID list.
- Added Unbreaking, Mending, standard sword enchantment, and compatible Advanced
  Mending support.
- Added material-specific off-hand damage and flat attack-speed bonuses.
- Added Wood, Stone, Gold, Diamond, Steel, Umbrium, Silver, Myrmex, Stinger,
  elemental Dragonbone, Living, and Sentient effects.
- Added parasite-only Living-to-Sentient evolution by default with a config switch.
- Added the Diamond Defender creative-tab icon and enabled enchantment books.
- Kept enchantment descriptions on books instead of applied Defenders.
- Added normal shield-disabling behavior without allowing projectile blocking.
- Preserved full normal attacks and uninterrupted mining while guarding.
- Fixed vanilla-style main-hand attack-speed display.
- Added optional Forgotten Nunchakus continuous-attack compatibility while
  raising, holding, and lowering a Defender.
