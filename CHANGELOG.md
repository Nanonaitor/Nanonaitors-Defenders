# Changelog

## 1.1.1 - Initial Forge 26.1.2 release

- Ported the Defender combat system to Minecraft 26.1.2 and Forge 64.0.12.
- Added Wood, Stone, Copper, Gold, Iron, Diamond, and Netherite Defenders.
- Added perfect parries, continued melee guarding, configurable movement and
  durability costs, and off-hand damage and attack-speed bonuses.
- Added Footwork, Reprisal, Finesse, Reflexes, Deflection, and 6th Sense.
- Added compatible vanilla weapon, Unbreaking, and Mending enchantments.
- Added the Diamond Defender creative-tab icon.
- Added vanilla-style Netherite smithing-table progression that preserves the
  Diamond Defender's data and enchantments.
- Kept projectile damage outside the base Defender blocking mechanic.
- Added dedicated client input handling so normal mining can continue while
  raising and lowering a Defender.
